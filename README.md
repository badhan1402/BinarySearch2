# Binary_search_2

 Modify the Binary Search function from  Binary_search_1 so that if the value cannot be found, it will return the index of the closest number smaller than the value.<br>
 Sample Input:<br>
 7<br>
 1 5 9 13 18 22 35<br>
 13<br>
 Sample Output:<br>
 3<br>
 Sample Input:<br>
 7<br>
 1 5 9 13 18 22 35<br>
 19<br>
 Sample Output:<br>
 4
